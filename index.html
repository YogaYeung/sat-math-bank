二、题目 + 详解

Q1

What is 10% of 470? A) 37 B) 47 C) 423 D) 460

答案 B。 10% × 470 = 0.1 × 470 = 47。

Q2

4x + 6 = 18. Which equation has the same solution? A) 4x = 108 B) 4x = 24 C) 4x = 12 D) 4x = 3

答案 C。 两边同减 6：4x = 18 − 6 = 12。同解变形即 4x = 12。

Q3

租冲浪板：固定服务费 $25 + 每小时 $10。租 t 小时，最多花 $75，求不等式。 A) 10t ≤ 75 B) 10 + 25t ≤ 75 C) 25t ≤ 75 D) 25 + 10t ≤ 75

答案 D。 总花费 = 25 + 10t，不超过 75 → 25 + 10t ≤ 75。

Q4

The function g is defined by g(x) = x² + 9. For which value of x is g(x) = 25? A) 4 B) 5 C) 9 D) 13

答案 A。 x² + 9 = 25 → x² = 16 → x = ±4。选项中为 4。

Q5

一个公平 14 面骰子（标 1~14），掷一次，求掷出 2 的概率。 A) 1/14 B) 2/14 C) 12/14 D) 13/14

答案 A。 1 个有利结果 / 14 个总结果 = 1/14。

Q6

打印机每分钟印 42 张海报，求每小时印多少张（学生填空题）。

答案 2,520。 42 × 60 = 2,520 张/小时。

Q7

f(x) = 7x + 2. 求 x = 4 时 f(x) 的值（填空题）。

答案 30。 f(4) = 7×4 + 2 = 28 + 2 = 30。

Q8

老师出一份满分 70 分的作业，由 1 分题（x 道）和 3 分题（y 道）组成，求方程。 A) 4xy = 70 B) 4(x+y) = 70 C) 3x + y = 70 D) x + 3y = 70

答案 D。 1 分题贡献 x 分，3 分题贡献 3y 分，总分 x + 3y = 70。

Q9

直角三角形 LMN ∽ PQR，L↔P、M↔Q 对应。∠M = 53°，求 ∠Q。 A) 37° B) 53° C) 127° D) 143°

答案 B。 相似三角形对应角相等，∠Q = ∠M = 53°。

Q10

y = -3x；4x + y = 15. 方程组的解为 (x, y)，求 x。 A) 1 B) 5 C) 15 D) 45

答案 C。 代入：4x + (−3x) = 15 → x = 15。

Q11 ⚠️依赖原图（散点图）

给定散点图，问哪条直线模型最合适。 A) y = -1.9x − 10.1 B) y = -1.9x + 10.1 C) y = 1.9x − 10.1 D) y = 1.9x + 10.1

答案 B。 散点呈负相关（斜率为负，排除 C/D），且 y 轴截距为正 → y = −1.9x + 10.1。

Q12 ⚠️依赖原图（三次函数图像）

f(x) = ax³ + bx² + cx + d 的图像已给出，问 f(x) = 0 有几个解（即图像与 x 轴交点数）。 A) One B) Two C) Three D) Four

答案 C（Three）。 图像与 x 轴有 3 个交点，故有 3 个实根。

Q13

Vivian 花 $71 买派对帽和纸杯蛋糕。每包帽子 $3，每个蛋糕 $1。买了 10 包帽子，问买了几个蛋糕（填空题）。

答案 41。 帽子花 10 × 3 = $30，剩 71 − 30 = $41，每个蛋糕 $1 → 41 个。

Q14

z² + 10z − 24 = 0. 求其中一个解（填空题）。

答案 2 或 −12。 因式分解 (z + 12)(z − 2) = 0 → z = 2 或 z = −12。填任一即可。

Q15

细菌培养：初始观测 300,000 个/毫升，每 3 小时翻倍。15 小时后多少个/毫升？ A) 1,500,000 B) 2,400,000 C) 4,500,000 D) 9,600,000

答案 D。 15 ÷ 3 = 5 次翻倍 → 300,000 × 2⁵ = 300,000 × 32 = 9,600,000。

Q16

Which expression is equivalent to 6x⁸y² + 12x²y² ? A) 6x²y²(2x⁶) B) 6x²y²(x⁴) C) 6x²y²(x⁶ + 2) D) 6x²y²(x⁴ + 2)

答案 C。 提公因式 6x²y²：6x²y²(x⁶ + 2)。验证：6x²y²·x⁶ = 6x⁸y²，6x²y²·2 = 12x²y²。✓

Q17

某社区 = 2 公顷公园 + 35 公顷住宅区，树木总数 3,934，方程 2x + 35y = 3,934。求 x 的最佳解释。 A) 公园每公顷平均树木数 B) 住宅区每公顷平均树木数 C) 公园树木总数 D) 住宅区树木总数

答案 A。 2（公园公顷数）× x = 公园树木总数，故 x = 公园每公顷平均树木数。

Q18 ⚠️依赖原图（直线关系图）

图示 Simone 可买 A 公司股票数 x 与 B 公司股票数 y 的关系，求方程。 A) y = 8x + 12 B) 8x + 12y = 480 C) y = 12x + 8 D) 12x + 8y = 480

答案 B。 由图两截距：x 轴截距 60（8×60=480）、y 轴截距 40（12×40=480），符合 8x + 12y = 480（总预算 $480，A 股 $8/股，B 股 $12/股）。

Q19

圆 A 半径 3n，圆 B 半径 129n（n 为正常数）。圆 B 面积是圆 A 的几倍？ A) 43 B) 86 C) 129 D) 1,849

答案 D。 面积比 = 半径比的平方 = (129n / 3n)² = 43² = 1,849。

Q20

频数表汇总 57 个数据值（数据值: 频数）：6:3, 7:5, 8:12, 9:?, 10:9, 11:11, 12:9, 13:0, 14:6。求数据集中的最大值（填空题）。

答案 14。 频数非 0 的最大数据值为 14（频数 6；13 的频数为 0 不计）。

Q21

xy 平面中一圆，直径端点为 (2, 4) 和 (2, 14)，方程 (x−2)² + (y−9)² = r²，求 r。 A) 4 B) 4.5 C) 5 D) 10

答案 C。 直径长 = 14 − 4 = 10 → 半径 r = 5。

Q22

∠R = 2π/3 弧度，∠T 比 ∠R 大 π/3 弧度（注：原题增量为 5π/12）。求 ∠T 的度数。 A) 75 B) 120 C) 195 D) 390

答案 C。 ∠R = 2π/3 = 8π/12，∠T = 8π/12 + 5π/12 = 13π/12。换算度数：13π/12 × (180/π) = 195°。

Q23

某镇面积 4.36 平方英里，求平方码数（1 mile = 1,760 yards）。 A) 404 B) 7,674 C) 710,459 D) 13,505,536

答案 D。 1 平方英里 = 1,760² = 3,097,600 平方码。4.36 × 3,097,600 = 13,505,536。

Q24

直线 h 过点：(18,130), (23,160), (26,178)。直线 k 是 h 向下平移 5 个单位。求 k 的 x 截距。 A) (−26/3, 0) B) (−9/2, 0) C) (−11/3, 0) D) (−17/6, 0)

答案 D。 斜率 = (160−130)/(23−18) = 30/5 = 6。h：130 = 6×18 + b → b = 22，即 y = 6x + 22。下移 5：y = 6x + 17。令 y = 0 → x = −17/6。

Q25

xy 平面中 y = −x² + 9x − 100 与直线 y = c 恰好交于一点，求 c。 A) −481/4 B) −100 C) −319/4 D) −9/2

答案 C。 恰好一个交点 = 直线与抛物线相切 = 抛物线顶点值。配方：y = −(x − 9/2)² − 319/4，最大值 = −319/4 → c = −319/4。

Q26

2x + 3y = 7；10x + 15y = 35. 两式实为同一直线（第二式 = 第一式 ×5）。对每个实数 r，下列哪个点在该直线上？ A) (r/5 + 7, −r/5 + 35) B) (−3r/2 + 7/2, r) C) (r, 2r/3 + 7/3) D) (r, −2r/3 + 7/2)

答案 B。 取 y = r，由 2x + 3y = 7 → x = (7 − 3r)/2 = −3r/2 + 7/2，得点 (−3r/2 + 7/2, r)。

Q27

等边三角形周长 624 厘米，高为 k√3 厘米（k 为常数），求 k（填空题）。

答案 104。 边长 = 624 ÷ 3 = 208。等边三角形高 = (√3/2) × 边长 = (√3/2) × 208 = 104√3 → k = 104。
